# 白内障（Cataract）

## 疾病概述
白内障是晶状体混浊导致视力下降的疾病，是全球首位可逆致盲原因。约占全球盲的 33%。按病因分为年龄相关性（最常见，老年性）、先天性、外伤性、代谢性（糖尿病）、药物性（皮质类固醇）等。晶状体混浊可位于核、皮质或后囊下。

## 危险因素
- 年龄（最主要，60岁以上发病率急增）
- 糖尿病（加速晶状体混浊）
- 紫外线长期暴露（核性白内障）
- 长期全身或局部使用糖皮质激素（后囊下白内障）
- 外伤、眼内炎症
- 吸烟（核性白内障风险增加）
- 高度近视（核性白内障风险）

## 临床症状
- 无痛性进行性视力下降（最主要症状）
- 眩光和光晕（尤其夜间驾驶，后囊下型明显）
- 对比敏感度下降
- 单眼复视（皮质型）
- 近视度数加深（核性白内障——晶状体核屈光力增加）
- 色觉改变（晶状体核变黄/棕，影响蓝色感知）
- 白瞳症（先天性或成熟期）

## 眼底特征
- 白内障本身为前段病变，眼底可见度因混浊程度而异
- 轻度：眼底可见，但图像欠清晰
- 重度：眼底无红光反射（leukocoria），眼底不可窥见
- 眼底检查前需评估晶状体透明度，必要时B超评估玻璃体和视网膜

## 建议检查项目
- 视力（最佳矫正视力 BCVA）
- 裂隙灯检查（混浊程度、位置、类型）
- 眼压（排除继发性青光眼）
- 散瞳眼底检查（评估伴随眼底疾病）
- 角膜内皮细胞计数（术前必查）
- 眼轴和角膜曲率（IOL 度数计算）
- 视力潜力测试（BAT眩光测试）

## 分级（LOCS III）
| 类型 | 描述 |
|------|------|
| 核性（NC） | 晶状体核颜色和混浊（NC1-NC6） |
| 皮质性（C） | 楔形皮质混浊 |
| 后囊下（PSC） | 后囊下颗粒状混浊，近轴光症状明显 |

## 转诊标准
- 视力 <0.3 或视力下降影响日常生活（驾驶、阅读）
- 眩光严重影响功能
- 成熟/过熟白内障（膨胀期需排除继发性青光眼）
- 糖尿病患者需检查眼底，白内障影响眼底观察时应考虑白内障手术

## 治疗原则
- **手术为唯一有效治疗**（无药物可逆转混浊）
- 超声乳化白内障摘除术 + 人工晶状体植入（IOL）为金标准
- 手术时机：视力下降影响功能即可考虑，无需等到"成熟"
- 注意合并眼底疾病（DR、AMD、青光眼）会影响术后视力预期
