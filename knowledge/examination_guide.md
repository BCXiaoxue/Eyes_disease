# 眼科检查项目指南

## 常用眼科检查项目说明

### 视力检查
- **远视力（BCVA）**：Snellen视力表，最佳矫正视力 ≥1.0 为正常
- **近视力**：阅读视力，评估中心视力精细功能
- **对比敏感度**：评估青光眼、白内障、视网膜病变的早期变化

### 眼压测量（Tonometry）
- **Goldmann压平眼压计**：金标准（正常范围 10-21 mmHg）
- **非接触眼压计（NCT）**：筛查用途
- **角膜厚度影响**：薄角膜低估眼压，厚角膜高估眼压（每偏差50μm约调整3 mmHg）
- 青光眼诊断需结合中央角膜厚度（CCT）解读眼压

### 裂隙灯检查（Slit-lamp）
- 评估前节：角膜、虹膜、晶状体（白内障分级）
- 前房深度和形态（闭角青光眼筛查）
- 房角镜检查：区分开角 vs. 闭角型青光眼

### 散瞳眼底检查
- **直接检眼镜**：小瞳可行，但视野有限
- **间接检眼镜**：大视野，适合周边视网膜评估
- **散瞳眼底照相**：记录和比较（非散瞳照相用于筛查）
- **超广角眼底照相（Optos/Clarus）**：单次捕获80-200°视野

### 光学相干断层扫描（OCT）
- **黄斑OCT（Macular OCT）**：评估黄斑厚度图、层间结构、积液、CNV
  - 正常黄斑中心凹厚度：~250μm
  - DME：黄斑增厚 >250μm 并有积液
  - AMD：PED、subretinal fluid、DRIL（脱落指数）
- **视神经OCT（RNFL OCT）**：
  - 环形扫描RNFL厚度（正常均值 ~100μm）
  - 神经节细胞复合体（GCC）分析
  - 用于青光眼诊断和随访
- **OCTA（OCT血管造影）**：无创血流成像，评估CNV、DR毛细血管层

### 荧光素眼底血管造影（FFA）
- 评估视网膜血管通透性、无灌注区、新生血管、CNV
- 用于DR分期、AMD分型（CNV分类）、静脉阻塞评估
- 需静脉注射荧光素钠，有过敏风险

### 视野检查（Perimetry）
- **Humphrey自动视野计（24-2/30-2）**：青光眼标准检查
- **Goldmann手动视野计**：周边视野评估
- **正常视野**：颞侧90°，鼻侧60°，上60°，下75°
- 青光眼典型缺损：鼻侧阶梯、弓形暗点、管状视野

### B超（眼超声检查）
- 适应证：屈光间质混浊（白内障致底无法直视）
- 评估：玻璃体状态、视网膜脱离、后巩膜葡萄肿、眼内肿物
- 眼轴测量（A超）：白内障术前IOL计算

### Amsler方格表
- 家庭自我监测黄斑功能
- 发现视物变形（metamorphopsia）：提示湿性AMD、ERM、CME
- 正常：格线直、等间距、中心点清晰
- 异常：直线弯曲、格格不等、中心暗点

## 检查项目选择建议

| 怀疑疾病 | 首选检查 |
|----------|----------|
| 糖尿病视网膜病变 | 散瞳眼底 + OCT黄斑 + HbA1c |
| 青光眼 | 眼压 + OCT-RNFL + 视野 + 前房角镜 |
| 湿性AMD | OCT黄斑 + FFA/OCTA |
| 白内障 | 裂隙灯 + 视力 + 散瞳眼底 + 角膜内皮 |
| 高血压视网膜病变 | 散瞳眼底 + 血压 + 肾功 |
| 病理性近视 | OCT黄斑 + 眼轴 + 超广角眼底照 |
