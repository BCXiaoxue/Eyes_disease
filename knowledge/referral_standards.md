# 眼科转诊标准与紧急程度分级

## 紧急程度分级

### 🔴 急诊（立即/当天）
以下情况危及视功能，需当天或立即急诊就诊：

1. **急性闭角型青光眼发作**：眼痛剧烈、恶心呕吐、虹视、眼压急性升高
2. **视网膜脱离**（有症状——新发大量飞蚊/闪光/幕布感）：黄金时间 24 小时
3. **玻璃体积血**（突发视力急剧下降，眼底无红光反射）
4. **视网膜中央动脉阻塞（CRAO）**：无痛性突发单眼失明，90 分钟内溶栓/降眼压
5. **恶性高血压**（血压 >180/120 + 视乳头水肿 + 出血）
6. **眼外伤**（穿通伤、化学伤）
7. **急性视神经炎**（突发视力下降伴眼动痛）

### 🟠 紧急（1-7天内）
1. **增殖性糖尿病视网膜病变（PDR）**或重度NPDR
2. **新发湿性AMD**（新出现视物变形）：尽早抗VEGF
3. **视网膜分支动脉阻塞（BRAO）**
4. **无症状视网膜裂孔（马蹄孔）**：1-2天内激光封闭
5. **myopic CNV**（病理性近视 + 视物变形/出血）
6. **视乳头水肿**（颅内压升高？）：需排除颅内病变

### 🟡 择期（2-4周内）
1. **白内障**（视力影响功能 BCVA <0.3 或眩光严重）
2. **中期AMD**（大软性玻璃膜疣 >125μm）
3. **可疑青光眼**（C/D比扩大 >0.6、RNFL缺损、眼压持续偏高）
4. **黄斑水肿**（非急性，OCT发现）
5. **格子样变性**（无孔、无症状）

### 🟢 常规（1-3月内）
1. **轻中度NPDR**（血糖控制后定期随访）
2. **早期AMD**（小玻璃膜疣）
3. **高眼压症**（眼压21-26 mmHg，无视神经损害）
4. **干眼、结膜炎等外眼疾病**

## 转诊信息模板
转诊时应提供以下信息：
- 患者基本信息（年龄、性别、系统性疾病史）
- 主诉和发病时间
- 初步眼科检查结果（视力、眼底描述）
- AI辅助诊断结果（供参考）
- 紧急程度标注

## 高危患者眼科随访频率建议

| 疾病 | 随访频率 |
|------|----------|
| 轻度NPDR | 每12月 |
| 中度NPDR | 每6月 |
| 重度NPDR | 每3月 |
| PDR（治疗后稳定） | 每2-3月 |
| 青光眼（稳定） | 每6月 |
| 湿性AMD（抗VEGF治疗中） | 每4-8周（按T&E方案） |
| 干性AMD中期 | 每6-12月 |
| 病理性近视（无并发症） | 每年 |

## 眼科急症红旗征象（Red Flags）
出现以下任一症状，应立即就医：
- ⚡ 突然失明或视力急剧下降
- ⚡ 大量飞蚊突然增多（特别是伴闪光感）
- ⚡ 视野中出现幕布/阴影
- ⚡ 剧烈眼痛伴头痛/恶心
- ⚡ 外伤后视力下降或异物感
- ⚡ 视物变形（直线弯曲）突然加重
